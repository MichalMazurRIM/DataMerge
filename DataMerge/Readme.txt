# DataMerge 

Proste narzędzie do łączenia wielu plików CSV i XLSX w jeden raport.  

Program:
• wczytuje wszystkie pliki z wybranego folderu  
• normalizuje kolumny tak, żeby wszystkie miały taki sam układ  
• łączy dane w jeden raport  
• usuwa duplikaty  
• zapisuje wynik do pliku XLSX o nazwie „raport_zbiorczy.xlsx”

To małe, praktyczne narzędzie, które możesz dać klientowi lub używać w pracy.

---

## Jak używać

1. Uruchom program `DataMergePro.exe`.
2. W oknie wybierz folder z plikami CSV/XLSX.
3. Kliknij **Start**.
4. W tym samym folderze pojawi się plik:
   - `raport_zbiorczy.xlsx`

---

## Wymagania

Wersja exe działa na:
• Windows 10  
• Windows 11
